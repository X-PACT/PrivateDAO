If you discover a security issue, please report it privately to the maintainers before public disclosure.

- Preferred: GitHub Security Advisory (private report)
- Fallback contact: open an issue marked `security` without exploit details and request a private channel.

## Scope

Security reports may include:
- On-chain authorization flaws
- Treasury execution bypasses
- Commit-reveal integrity flaws
- Replay or signature misuse

Please include clear reproduction steps and affected commit hash.
