# Terms Of Public Source Review

PrivateDAO is public and open source where the repository license applies. It is also a review surface for accountability, judging, security research, and ecosystem coordination. By using this repository as a reviewer, evaluator, security researcher, partner, or potential integrator, you should respect both the open-source license and the reserved official-use rights.

## Allowed Without Prior Coordination

- Read the code and documents.
- Evaluate the project for judging, diligence, audit scoping, or research.
- Cite public routes, transaction hashes, and proof endpoints with attribution.
- Report security issues privately.
- Discuss official integration with the PrivateDAO team.

## Not Allowed Without Written Coordination

- Reuse PrivateDAO branding, product identity, official site layout, or founder materials outside the license and official project boundary.
- Publish a derivative that appears to be an official PrivateDAO deployment.
- Copy review packets, service packaging, or roadmap language into a competing product claim.
- Use PrivateDAO digital evidence as proof for another project.
- Offer commercial PrivateDAO-derived services or pilots.
- Circumvent license, payment, or access controls in production service packages.

## License Boundary

Some source files are licensed under AGPL-3.0-or-later. Those rights and duties remain where they apply. The repository also includes reserved brand, documentation, media, evidence, roadmap, service packaging, and official deployment material that is not granted for unrestricted official or commercial reuse by repository visibility alone.

## Contact

For official reuse, security review, partner review, pilot access, or institutional coordination, use the official PrivateDAO channels linked from `https://privatedao.org/`.
