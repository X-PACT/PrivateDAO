What the UI owns

Dashboard is part of the public product surface. It is responsible for execution visibility, logs, and diagnostics, while repo-only tools remain outside the buyer-facing runtime.

UI Full

Start with Solflare, Phantom, or Backpack and activate the live governance flow.

Open the command center and stage governance actions from the product shell.
