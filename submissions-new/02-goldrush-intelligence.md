## Project description

PrivateDAO uses structured data from GoldRush and Dune Sim to make treasury decisions safer before signatures happen. Instead of raw RPC output, operators see risk hints, stablecoin concentration, holdings snapshots, and transaction previews directly in the decision flow.

## How this track is used

- Live server proxy for GoldRush queries.
- Dune Sim transaction augmentation.
- Reviewer-readable intelligence cards in product, not raw JSON only.
