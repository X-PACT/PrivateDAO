## Project description

PrivateDAO turns swap and rebalance operations into governance-safe routes. Operators can preview route quality and quote posture inside the same product shell used for proposals and approvals, then move into execution and proof without context loss.

## How this track is used

- Live server endpoint for route preview.
- Quote summary + top route split in product UI.
- Direct reviewer packet for route rationale and traceability.
