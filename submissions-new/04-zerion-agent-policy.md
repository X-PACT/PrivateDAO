## Project description

PrivateDAO frames autonomous execution as policy-bound, not god-mode. The agent lane locks actions by chain, spend scope, and expiry, then ties operator review and proof continuity to the same governance shell.

## How this track is used

- Policy templates for bounded execution.
- Live server-side wallet portfolio summary.
- Judge and proof routes point directly to the policy packet.
