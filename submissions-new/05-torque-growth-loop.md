## Project description

PrivateDAO uses Torque as a measurable retention layer attached to real actions, not vanity clicks. Governance actions, billing signatures, learning completions, and private treasury execution are emitted as custom events through a server-side relay.

## How this track is used

- Live forwarding to Torque ingestion API (`/events`).
- `private_treasury_execution` event shape supported directly in product.
- Delivery receipts shown in UI for reviewer traceability.
