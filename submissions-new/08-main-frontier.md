## Project description

PrivateDAO is an operating system for Solana organizations that need governed actions, private-ready settlement rails, structured intelligence, and auditable proof in one product. The experience is wallet-first for normal users and evidence-first for reviewers.

## Why this wins technically

- one operating flow: connect, review, sign, verify
- one proof continuity path across routes
- one reviewer fast path from Judge into each integration lane
- live server-side lanes for intelligence, routing preview, portfolio policy checks, and growth events
