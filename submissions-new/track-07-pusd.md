## Short pitch

PrivateDAO uses PUSD as a core stable operating mode for payroll, grants, and reward pools. The route is wallet-first for users and proof-linked for judges, with the same treasury flow tied to billing rehearsal and reviewer packets.

## How we used the track

- Added dedicated PUSD route:
  - `/services/pusd-stablecoin`
- Wired PUSD route directly into judge and proof fast paths.
- Kept billing execution lane and packet continuity attached to the same operating path.
- Preserved wallet-first execution and governance context in one surface.
