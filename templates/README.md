# PrivateDAO Templates

This directory holds reusable product starters.

Main bootcamp set:

- [`frontend-solana-bootcamp/`](./frontend-solana-bootcamp/)

Zero-to-run path:

1. `git clone https://github.com/X-PACT/PrivateDAO.git`
2. `cd PrivateDAO`
3. `npm install`
4. `npm run dev`
5. Open the matching live route:
   - Wallet starter -> `/start/`
   - Governance starter -> `/govern/`
   - Runtime starter -> `/dashboard/`
   - Private payment starter -> `/security/`

These templates are designed to be copied into a React / Next.js shell, but the fastest way to see them working is to run the live product route they map to inside this repository.
