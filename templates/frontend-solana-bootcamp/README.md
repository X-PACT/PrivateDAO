# PrivateDAO Frontend Solana Bootcamp Templates

These starter templates are plug-and-play references for the `/learn` bootcamp.

Templates:

- `wallet-connect-starter/WalletConnectStarter.tsx`
  - Wallet-first entry shell
  - GitHub-facing alias: `wallet-template`
- `proposal-ui-starter/ProposalUiStarter.tsx`
  - Governance proposal and voting surface
  - GitHub-facing aliases: `governance-template`, `dao-ui-template`
- `runtime-activity-starter/RuntimeActivityStarter.tsx`
  - Runtime state, tx hash, and diagnostics widget
  - GitHub-facing alias: `runtime-template`
- `private-payment-starter/PrivatePaymentStarter.tsx`
  - Private payout and proof-linked payment surface
  - GitHub-facing alias: `payment-template`

How to use:

1. Open the matching lecture in `/learn`.
2. Open the matching live route in the product.
3. Copy the starter component into your app shell.
4. Replace mock callbacks with your real wallet and route handlers.
5. Verify the result on Devnet from the same lecture corridor.
