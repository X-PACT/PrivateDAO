# Private Payment Starter

Use this starter to teach:

- private payroll and grant request UX
- gaming reward execution UX
- ZK / REFHE / MagicBlock explanation in product language
- proof CTA and chain verification CTA placement

Expected outcome:

- the user understands what stayed private, what became public, and where the verification evidence lives.

Plug-and-play run path:

1. `git clone https://github.com/X-PACT/PrivateDAO.git`
2. `cd PrivateDAO`
3. `npm install`
4. `npm run dev`
5. open `https://localhost:3000/security/`

Fastest proof:

- open `https://privatedao.org/security/`
- then verify through `https://privatedao.org/judge/`
