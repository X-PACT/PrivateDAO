# Proposal UI Starter

Use this starter to teach:

- proposal creation
- commit-reveal lifecycle
- execution status
- proof CTA placement

Expected outcome:

- the user sees governance as a guided lifecycle, not a protocol debugging session.

Plug-and-play run path:

1. `git clone https://github.com/X-PACT/PrivateDAO.git`
2. `cd PrivateDAO`
3. `npm install`
4. `npm run dev`
5. open `https://localhost:3000/govern/`

Fastest proof:

- open the live route `https://privatedao.org/govern/`
- compare the starter component with the live governance lane
