# Runtime Activity Starter

Use this starter to teach:

- transaction visibility
- status after wallet action
- verification CTA placement
- runtime-aware UX

Expected outcome:

- the user can tell what happened, where to verify it, and whether retry or waiting is the correct next step.

Plug-and-play run path:

1. `git clone https://github.com/X-PACT/PrivateDAO.git`
2. `cd PrivateDAO`
3. `npm install`
4. `npm run dev`
5. open `https://localhost:3000/dashboard/`

Fastest proof:

- open the live route `https://privatedao.org/dashboard/`
- then continue into `https://privatedao.org/diagnostics/`
