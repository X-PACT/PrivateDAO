# Wallet Connect Starter

Use this starter when teaching a Web2 frontend builder how to begin from:

- wallet connection
- signer visibility
- one-click route entry

Expected outcome:

- the user connects a Devnet wallet
- sees signer context immediately
- moves into governance or treasury work without terminal steps

Plug-and-play run path:

1. `git clone https://github.com/X-PACT/PrivateDAO.git`
2. `cd PrivateDAO`
3. `npm install`
4. `npm run dev`
5. open `https://localhost:3000/start/`

Fastest proof:

- open the live route `https://privatedao.org/start/`
- compare this starter component with the live route shell
